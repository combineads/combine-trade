# T-23-005 Tauri App Icon Replacement

## Goal
Replace the placeholder PNG icons in `apps/desktop/src-tauri/icons/` with Combine Trade logo-derived raster icons, and add `.icns` (macOS) and `.ico` (Windows) formats.

## Why
The desktop app currently uses Tauri default placeholder icons. Replacing them with the branded logo ensures the app is recognizable in the dock, taskbar, and OS app switcher.

## Inputs
- Generated PNG icons from T-23-001 (32, 128, 256, 512, 1024px)
- `apps/desktop/src-tauri/tauri.conf.json` (bundle.icon config)
- `docs/assets/logo/icon-dark.svg` (source)

## Dependencies
- T-23-001 (icon generation script produces the raster files)

## Expected Outputs
- Updated `apps/desktop/src-tauri/icons/` with branded PNGs + `.icns` + `.ico`
- Updated `tauri.conf.json` bundle.icon array if new formats added

## Deliverables
- Replace existing PNGs in `apps/desktop/src-tauri/icons/`:
  - `32x32.png` (32×32)
  - `128x128.png` (128×128)
  - `128x128@2x.png` (256×256)
  - `icon.png` (512×512 or 1024×1024)
- Add `icon.icns` (macOS app bundle)
- Add `icon.ico` (Windows executable)
- Update `tauri.conf.json` `bundle.icon` to include `.icns` and `.ico`:
  ```json
  "icon": [
    "icons/32x32.png",
    "icons/128x128.png",
    "icons/128x128@2x.png",
    "icons/icon.png",
    "icons/icon.icns",
    "icons/icon.ico"
  ]
  ```

## Constraints
- Icon must use the dark-background variant (dark bg with green/red crossing lines)
- `.icns` must contain standard Apple icon sizes
- `.ico` must contain 16, 32, 48, 256px sizes
- Generated by T-23-001 script — this task places and configures them

## Steps
1. RED: Write test that verifies all required icon files exist in `src-tauri/icons/` with correct dimensions
2. GREEN: Run `generate:icons` and verify outputs land in correct locations; update `tauri.conf.json`
3. REFACTOR: Verify `cargo tauri build --debug` can read the icons

## Acceptance Criteria
- All icon files in `apps/desktop/src-tauri/icons/` are branded (not Tauri default)
- `tauri.conf.json` references all icon formats
- `file` command confirms correct dimensions for each PNG
- `.icns` and `.ico` files are valid

## Validation
```bash
file apps/desktop/src-tauri/icons/32x32.png | grep "32 x 32"
file apps/desktop/src-tauri/icons/128x128.png | grep "128 x 128"
file apps/desktop/src-tauri/icons/icon.png | grep -E "(512 x 512|1024 x 1024)"
test -f apps/desktop/src-tauri/icons/icon.icns && echo "icns exists"
test -f apps/desktop/src-tauri/icons/icon.ico && echo "ico exists"
bun run typecheck
```

## Out of Scope
- System tray icons (T-23-006)
- WebView favicon (T-23-007)
- macOS DMG background image
