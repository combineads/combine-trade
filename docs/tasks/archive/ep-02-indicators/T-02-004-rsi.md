# T-02-004 indicators/rsi.ts — RSI(14) calculation

## Goal
Implement Relative Strength Index calculation wrapping `@ixjb94/indicators` IndicatorsSync.rsi(). Default period is 14 (industry standard).

## Why
RSI is used in Safety Gate filters (overbought/oversold rejection) and as a vectorization feature. It measures momentum strength on a 0-100 scale.

## Inputs
- `src/indicators/types.ts` (T-02-001) — RSIResult type (Decimal | null)
- `src/core/decimal.ts` — d() factory
- `@ixjb94/indicators` — IndicatorsSync.rsi(source, period)

## Dependencies
T-02-001 (indicators/types.ts)

## Expected Outputs
- `src/indicators/rsi.ts` — calcRSI() function
- Used by Safety Gate, vectorization

## Deliverables
- `src/indicators/rsi.ts`

## Constraints
- L2 module: imports from `core/` and `indicators/types`
- RSI range is 0-100 (enforce via clamping if library returns out-of-range)
- Default period = 14 (not from constants.ts — industry standard, hardcoded default)
- Return null when insufficient data (< period + 1 candles)

## Steps
1. Import IndicatorsSync from `@ixjb94/indicators`
2. Import d, Decimal from `@/core/decimal`
3. Define RSI_DEFAULT_PERIOD = 14
4. Create `calcRSI(closes: number[], period: number = RSI_DEFAULT_PERIOD): Decimal | null`:
   - Return null if closes.length < period + 1
   - Call IndicatorsSync.rsi(closes, period)
   - Take last non-NaN value
   - Clamp to [0, 100] range
   - Return as Decimal
5. Create `calcRSISeries(closes: number[], period: number = RSI_DEFAULT_PERIOD): Decimal[]`:
   - Full series output, filter NaN, clamp each value
6. Export calcRSI, calcRSISeries, RSI_DEFAULT_PERIOD
7. Write tests
8. Verify typecheck passes

## Acceptance Criteria
- calcRSI returns value between 0 and 100 (inclusive)
- Returns null when closes.length < period + 1
- RSI of strongly rising prices → high value (> 70)
- RSI of strongly falling prices → low value (< 30)
- Output is Decimal type
- `bun run typecheck` passes

## Test Scenarios
- calcRSI with 15+ steadily rising closes → returns Decimal > 70
- calcRSI with 15+ steadily falling closes → returns Decimal < 30
- calcRSI with 14 closes (period=14) → returns null (needs period+1)
- calcRSI with 15+ alternating rises/falls → returns value near 50
- calcRSI result is always between 0 and 100
- calcRSISeries returns array of Decimal values
- calcRSI with constant prices → returns d("50") or close to it (no change = neutral)

## Validation
```bash
bun run typecheck
bun test --grep "rsi"
```

## Out of Scope
- Stochastic RSI
- RSI divergence detection
