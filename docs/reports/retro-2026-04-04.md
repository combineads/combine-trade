## Patterns discovered

### [New] Scaffold 태스크 통합 가능
- **Observed in**: T-03-007, T-03-008, T-03-009
- **Category**: efficiency
- **Description**: 동일 패턴의 스캐폴드 어댑터 3개가 별도 태스크로 생성됨
- **Root cause**: 에픽의 task candidates가 1:1로 태스크에 매핑됨
- **Recommendation**: 동일 패턴 반복 시 하나의 태스크로 합치기. "OKX/Bitget/MEXC scaffold"를 1개 태스크로 처리하면 ~15% 시간 절감

### [New] 인터페이스 변경은 첫 태스크에 집중
- **Observed in**: T-03-001, T-03-004
- **Category**: scope-creep
- **Description**: ports.ts가 2개 태스크에서 각각 수정됨 (setLeverage, stop_market)
- **Root cause**: SL/leverage 요구사항이 base 태스크 시점에 완전히 파악되지 않음
- **Recommendation**: 인터페이스 변경이 필요한 경우 첫 태스크(base)에서 모두 처리. 후속 태스크는 인터페이스를 구현만 해야 함
